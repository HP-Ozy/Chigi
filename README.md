# 🏛️ Chigi

> Scopri cosa succede davvero nelle stanze del potere italiano — in modo semplice e veloce.

![chigi-demo-map-avvincente](https://github.com/user-attachments/assets/3cd76558-df99-4a99-af58-9a3c54597e44)

---

## Cos'è?

Un tool che raccoglie e presenta in chiaro le informazioni **pubbliche ma difficili da trovare** sul Parlamento italiano: voti, assenze, stipendi e molto altro.

Niente giornalismo, niente opinioni. Solo dati ufficiali, resi leggibili.


## Come si usa

```bash
# 1. Clona il progetto
git clone https://github.com/utente/chigi.git
cd chigi

# 2. Installa le dipendenze
pip install -r requirements.txt

# 3. Avvia
python app.py
```

Apri il browser su `localhost:5050` e inizia a esplorare.
